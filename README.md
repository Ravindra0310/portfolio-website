## ✨ Key Features

### 🎮 **2048 Tech Edition**
An custom-engineered version of the classic 2048 game built into the portfolio. Users merge technologies (HTML → CSS → JS → Kotlin → Android → Flutter → Compose) to reach the ultimate goal: **KMM 🚀**. 
*   Brand-matched neon glow for each tile.
*   Touch/Swipe support for mobile.
*   High-score persistence via `localStorage`.

### ⚡ **Motion & Interactivity**
*   **Custom Spring Cursor:** A neon ring with lag-behind physics that reacts to interactive elements.
*   **Typed Hero:** Dynamic typewriter effect showcasing lead technical roles.
*   **Decorative Layers:** Floating neon orbs, scanline overlays, and noise texture for a premium "dark room" feel.
*   **Animated Stats:** Scroll-triggered counters for years of experience and applications shipped.

### 🍱 **Portfolio Sections**
*   **01 — Hero:** High-impact introduction with quick contact details.
*   **02 — About:** Professional summary and high-level success metrics.
*   **03 — Selection:** Curated gallery of mobile ecosystems and technical experiments.
*   **04 — Experience:** Interactive timeline of 5+ years building Android apps for millions of users.
*   **05 — Toolkit:** Detailed tech stack visualization with skill progress indicators.
*   **06 — Play:** Integrated 2048 minigame.
*   **07 — Contact:** Clean contact portal with copy-to-clipboard email and social links.

## 🚀 Getting Started

### Prerequisites
*   [Node.js](https://nodejs.org) (v18+)
*   [npm](https://www.npmjs.com)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ravindra0310/portfolio-website.git
   cd portfolio-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup environment (Optional — for Contact Form):
   Follow the [EmailJS setup guide](https://www.emailjs.com/docs/sdk/installation/) and replace the placeholders in `src/App.tsx`.

4. Run locally:
   ```bash
   npm run dev
   ```

### Building for Production
```bash
npm run build
```
